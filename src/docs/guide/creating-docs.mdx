## 1. ローカルでの環境構築

ローカルで実行できるようにします。

```sh title=gitリポジトリをクローン
git clone https://github.com/sintaro/pj_code_recipes.git
```

必要モジュールをインストールし実行します。

```sh
cd pj_code_recipes
yarn install
yarn start
```

`http://localhost:8000`でローカルで実行が確認できます。
