## ⚡️ Getting started

1. Clone from the repo.

    ```sh
    git clone https://github.com/sintaro/pj_code_recipes.git
    ```

2. Start developing.

    ```sh
    cd pj_code_recipes
    yarn install
    yarn start
    ```

3. Are you ready for launch? 

    Your site is now running at `http://localhost:8000`
